# Числовой Хаос

**Жанр:** Логическая головоломка на поиск паттернов

---

## Суть

На поле разбросаны числа. Внутри этого набора спрятаны несколько числовых последовательностей,
перекрывающихся между собой. Игрок должен находить и удалять их, пока поле не очистится.

---

## Правила

1. Игрок выделяет 3 и более чисел, образующих корректную последовательность.
2. Выделенные числа удаляются с поля.
3. Одно число может входить в несколько последовательностей — выбор стратегии за игроком.
4. Победа — поле полностью очищено (или найдены все заложенные последовательности).

---

## Типы последовательностей

| Тип | Пример |
|-----|--------|
| Арифметическая (+n) | 2, 5, 8, 11 |
| Геометрическая (×n) | 2, 4, 8, 16 |
| Фибоначчи | 1, 1, 2, 3, 5, 8 |
| Чётные / нечётные | 2, 4, 6, 8 |
| Простые числа | 2, 3, 5, 7, 11 |
| Степени числа | 2, 4, 8, 16, 32 |

---

## Пример

Дано: `7, 3, 5, 10, 1, 8, 2, 4, 6, 9`

Варианты удаления:
- `10, 8, 2, 4, 6` — чётные числа
- `3, 5, 8` — последовательность Фибоначчи
- `2, 4, 6` — арифметическая (+2)

Порядок удаления влияет на доступные ходы.

---

## Параметры сложности

| Уровень | Поле | Последовательностей | Ходы | Время |
|---------|------|----------------------|------|-------|
| Лёгкий | 4×4 | 1–2, явные | без огр. | без огр. |
| Средний | 5×5 | 2–3, с перекрытием | 10 | 3 мин |
| Сложный | 6×6 | 3–4 + ложные числа | 6 | 2 мин |

---

## Особые механики (сложные уровни)

- **Ложные числа** — не входят ни в одну последовательность, отвлекают
- **Меняющиеся правила** — после каждого хода активный тип последовательностей меняется
- **Ограниченные ходы** — нужно оптимально выбрать порядок удалений
- **Комбо** — бонус за удаление двух последовательностей за один ход

---

## Прогрессия уровней

- Уровни 1–10: одна очевидная последовательность, есть подсказки
- Уровни 11–20: два-три перекрывающихся паттерна
- Уровни 21+: ложные числа, меняющиеся правила, комбо

---

## Подсказки

- Подсветка возможных последовательностей при долгом бездействии
- Явная подсказка с указанием одного валидного хода (с штрафом к очкам)
