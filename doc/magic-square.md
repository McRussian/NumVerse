# Магический Квадрат

**Жанр:** Классическая логическая головоломка

---

## Суть

Нужно расставить числа в квадратной сетке так, чтобы суммы по всем строкам, столбцам
и обеим диагоналям были одинаковыми (магическая константа).

---

## Правила

1. Дана сетка N×N с частично заполненными ячейками.
2. Нужно вставить оставшиеся числа из заданного набора.
3. Каждое число используется ровно один раз.
4. Победа — все суммы строк, столбцов и диагоналей равны магической константе.

---

## Пример — квадрат 3×3

Магическая константа: **15**

```
[2][7][6]
[9][5][1]
[4][3][8]
```

Проверка:
- Строки: 2+7+6=15, 9+5+1=15, 4+3+8=15 ✓
- Столбцы: 2+9+4=15, 7+5+3=15, 6+1+8=15 ✓
- Диагонали: 2+5+8=15, 6+5+4=15 ✓

---

## Режимы

| Режим | Описание |
|-------|----------|
| Классический | Числа 1..N², стандартный магический квадрат |
| Неполный | Половина ячеек заполнена, нужно найти остальные |
| Диагональный | Дополнительное условие — диагонали тоже равны (всегда) |
| Нестандартный | Числа не подряд (например, 2,4,6,8,...) |

---

## Параметры сложности

| Уровень | Размер | Заполнено | Подсказки |
|---------|--------|-----------|-----------|
| Лёгкий | 3×3 | 5 из 9 | есть |
| Средний | 4×4 | 4 из 16 | ограничены |
| Сложный | 5×5 | 0 из 25 | нет |

---

## Подсказки

- Показать магическую константу
- Подсветить строку/столбец, который нарушает условие
- Вставить одно число на место (штраф к очкам)
